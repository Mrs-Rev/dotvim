dotvim
======
My .vim configuration.

Installation:
-------------
Clone the .vim and vundle:

	git clone https://github.com/Mrs-Rev/dotvim.git ~/.vim
	git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Remove these files:

	cd ~
	rm .vimrc

Create symlinks:

	cd ~
	ln -s .vim/.vimrc .vimrc
